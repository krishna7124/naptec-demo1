NAPTEC Trust Hub

Project Summary — Client Handover Document

June 2026

Overview

NAPTEC Trust Hub is a fully live, professional website built for the National Accreditation and Professional Training Evaluation Council. It is publicly accessible and ready for use.

🌐 Live URL: https://krishna7124.github.io/naptec-demo1/

Pages

  • Home — Hero, services overview, process, stats, and testimonials
  • About — Mission, vision, core values, and governance structure
  • Process — Step-by-step accreditation journey
  • Standards — Quality frameworks and criteria
  • Services — Full list of accreditation and evaluation services
  • Directory — Searchable list of accredited organisations
  • Contact — Google Form for general enquiries
  • Apply — Google Form for accreditation applications

How It's Hosted

The website runs on GitHub Pages — a free, reliable hosting platform by GitHub. There are no monthly server costs. Any update pushed to the codebase is automatically live within 2 minutes.

Forms & Data Collection

Both the Contact and Apply forms are powered by Google Forms. Every submission is automatically saved to a Google Sheet, giving a permanent and searchable database of all enquiries and applications.

  • Unlimited submissions — no caps or fees
  • Accessible anytime from Google Sheets
  • Email notifications on every new submission (configurable)

Accredited Organisations Directory

The directory is managed through a simple data file (orgs.json). Adding a new accredited organisation requires editing one line in that file — no developer needed for routine updates.
